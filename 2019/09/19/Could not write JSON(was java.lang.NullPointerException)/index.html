每个实体类的属性的数据类型可能用了基本类型，也可能用了包装类型。

可能在处理业务的时候 null 值的处理方式，我们直接修改 bean 的字段类型为包装类，但是又没有把setter/getter方法重新生成。

就比如我的一个字段类型仅仅把属性的基本类型（如 double ）改为包装类型（如 Double ），而并没有将setter/getter的类型一起修改。
